# 💜 Portafolio - Deyanira Chacaliaza

Portafolio personal desarrollado con React, que muestra mi información profesional, habilidades y experiencia laboral.

## Instalación y uso

1. Clona el repositorio
```bash
git clone https://github.com/yamiletArias/portafolioCv-dy.git
cd portafolioCv-dy
```

2. Instala las dependencias
```bash
npm install
```

3. Inicia el servidor de desarrollo
```bash
npm start
```

4. Abre en tu navegador

http://localhost:3000
